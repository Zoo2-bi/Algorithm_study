# 📘 Algorithm Study

알고리즘 문제 해결 능력 향상을 위한 스터디입니다.
정해진 커리큘럼과 일정에 따라 매주 문제를 풀이하고 공유합니다.

작업 확인 방식으로는 풀이를 push 후 코멘트 작성, 디코, 만나서 회의 등이 가능할 수 있습니다.

---

## 📌 진행 방식

* 알고리즘 풀이 순서는 아래 커리큘럼을 따릅니다.
  👉 https://codingdodo.tistory.com/94

* 매주 정해진 문제를 각자 풀이 후 GitHub에 업로드합니다.

* 코드 리뷰 및 풀이 공유를 통해 다양한 접근 방식을 학습합니다.
* 단계별로 문제 난이도가 상승하니 처음에는 코드래빗을 사용하지 않을 것이지만 추후 탐색 부분 시작 시 코드래빗을 사용할 예정입니다.

---

## 📅 스터디 규칙

* 주어진 기간 내 문제 풀이 필수
* 최소 1문제 이상 풀이 및 업로드

**커밋 메시지 형식**

```
[문제번호] 문제이름
```

예시:

```
[1330] 두 수 비교하기
```

* 각자의 목표 단계를 나눈 후 서로 격려와 도움을 주는 스터디가 되었으면 합니다.
