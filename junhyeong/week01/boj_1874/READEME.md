## 💡 문제 설명

- 문제에 대한 간단한 요약 (내가 이 문제를 얼마나 파악하고 있는지 확인)

- 중복되지 않는 1이상 n이하인 n개의 정수들로 이뤄진 수열이 주어짐
- stack에 push하는 순서는 오름차순을 지킴
- 1부터 n까지의 숫자를 push, pop 연산을 통해 주어진 수열을 만들기

---

## 🚀 접근 방법

### 어떤 방식으로 접근했는지

#### 문제 조건에 집중 - push에 오름차순을 유지

- push할 때 오름차순을 유지했다면, pop할 때는 내림차순일 것이다.
- 이 때, 수열에서도 오름차순 부분이 있을 수 있는데, 그 차이는 1을 초과해야한다.
- 1차이가 나면서 오름차순인 경우에는, push할때 내림차순으로 넣어야만 가능한데 이는 문제의 조건을 위배하기 때문이다.

### 핵심 로직

- for문을 돌면서 입력 예제의 숫자가 stack에 있는지 확인
- 없다면 해당 수 만큼 push 후 pop
- 있다면 해당 수가 stack의 top과 같은지 확인
  - 같다면 pop
  - 다르다면 break

---

## ⏱️ 시간 복잡도

- 시간: O( n^ )
  - for문 내부의 includes() 메서드의 시간복잡도가 O(n)이므로, n^
- 공간: O( n )
  - stack의 최대길이가 n이므로 n

---

## 🌅 느낀 점

### 배운 점

- 시간복잡도 계산법
  - 단순히 중첩 for문이라고 해서 무조건 O(n^)인 것이 아님
  - 시간복잡도가 O(n)인 것이 중첩되었을때 n^

- 시간복잡도를 줄이도록 노력해보기
  - 내가 해결한 코드의 시간복잡도가 O(n^)인 것을 보고, 모범 답안 코드의 시간복잡도가 O(n)인 것을 보고 어떻게 하면 줄일 수 있을지 고민해보는 시간을 가질 수 있었다.
  - 자바스크립트의 내장 메서드인 inclues()를 사용한 탓이었다.

### 아쉬운 점

- includes() 메서드로 인하여 시간복잡도가 모범 답안 코드에 비해 더 크게 나왔다.

### 개선 포인트

- 불필요한 연산(includes() 메서드)을 줄여서 풀이해보는 방법을 시도해봐야겠다.
