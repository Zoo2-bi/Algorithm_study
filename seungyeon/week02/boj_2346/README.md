---

## 💡 문제 설명
<!--문제에 대한 간단한 요약 (내가 이 문제를 얼마나 파악하고 있는지 확인)-->

- 1번부터 N번까지의 풍선이 원형으로 놓임
- i번 풍선 왼쪽엔 i-1번 풍선, 오른쪽엔 i+1번 풍선
- 각 풍선엔 종이가 들어있음, 종이는 -N과 N 사이의 정수
- 양수면 오른쪽, 음수면 왼쪽으로 이동
- 이미 터진 풍선은 제외하고 이동
